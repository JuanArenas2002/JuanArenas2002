## About Me

Systems Engineering student at **Universidad Simón Bolívar** with hands-on experience building production-grade REST APIs and data processing pipelines in Python.

My focus is on **backend architecture** — designing systems that are maintainable, testable, and built to grow. I apply hexagonal architecture and domain-driven design principles even in academic and personal projects.

**What I bring:**
- Structured thinking: ports & adapters, clean dependency rules
- Data engineering: ETL pipelines, external API integration (Scopus, OpenAlex)
- API design: RESTful conventions, Pydantic validation, async FastAPI
- Testing mindset: unit-tested domain logic, isolated from infrastructure
